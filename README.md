# 📧 Email Testing

The demo credentials are intended only for testing the application's authentication.

If you want to test the **email notification feature**, please:

1. Register a new user account using a **valid email address**.
2. Log in with the newly created account.
3. Book any hotel.
4. The booking confirmation email will be sent to the registered email address.

> **Important:** Email notifications work only with valid email addresses. The demo credentials will **not** receive emails because the listed email IDs are placeholder accounts.
